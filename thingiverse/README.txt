# Summary

I'm using /u/johnniewhiskey 's [Cable Chain mod](https://www.thingiverse.com/thing:2920060) on the Ender 3 Pro.  Specifically, the v3 type links.  

This package tailors the [10 degree bed mount clip](https://www.thingiverse.com/thing:3162517) by Ryguitar07's to be compatible with the new links.  I have also shortened the offset of the original cableChain_40MountSide_v2 to compensate for the space taken by the angled mount. 


# Post-Printing

The bed clip just goes over the existing stock base mount.  Unclip the stock top and remove the zip tie.  This new clip should snap in place.  Ideally re-secure with a zip tie.

The side mount is too short to be a press-fit like the original.  You'll need to take off the end cover from the 4040 rail and slide this piece in.
